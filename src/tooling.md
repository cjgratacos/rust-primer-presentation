# Tooling
 
## Cargo

Cargo is the Rust package manager.
Cargo:
- Downloads your Rust package's dependencies.
- Compiles your packages.
- Makes distributable packages.
- Uploads them to crates.io (the Rust communityâ€™s package registry)

Example of commads:

```sh
# Create project
$ cargo new hello_world

# Build project
$ cargo build

# Run project
$ cargo run

# Test project
$ cargo test

```
